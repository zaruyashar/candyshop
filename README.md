# 🍬 CANDYSHOP

A visually striking, code-first MVC application developed as Project #2 for the SoftITo Backend Development Program. It features a fully dynamic contact system equipped with sliding window rate limiting (3 messages per 5 minutes) and a highly polished, elegant administrative dashboard.

## 🛠️ Tech Stack

* **C# .NET Core**
* **ASP.NET Core MVC**
* **Entity Framework Core (Code-First)**
* **T-SQL**
* **SweetAlert**
